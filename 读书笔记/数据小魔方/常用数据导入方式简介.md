## TXT文件导入
```r
# TXT读入
data<-read.table("C:\\Users\\Administrator\\Desktop\\myfile.txt",header=T)
```

## CSV文件导入
```r
# CSV数据读入
data<-read.csv("C:\\Users\\Administrator\\Desktop\\myfile.csv")
```
> 默认参数为header=T,即顶行为变量名称

## 剪切板直接复制
```r
# 直接复制
data <- read.table("clipboard", header = T, sep = '\t')
```
